# Claude Design — Curso do Zero ao Expert

Curso completo sobre o **Claude Design** (Anthropic Labs, `claude.ai/design` — lançado em 17/04/2026 sobre Claude Opus 4.7), no padrão visual **INEMA.CLUB**.

> Ferramenta colaborativa de criação visual: você **conversa** em vez de desenhar. Designs, protótipos, slides e peças de marketing gerados via chat, com identidade de marca aplicada automaticamente.

## Trilhas

| # | Trilha | Cor | Nível | Foco |
|---|--------|-----|-------|------|
| T1 | Fundamentos | Emerald | Iniciante | O que é, fluxo, primeiros designs |
| T2 | Intermediário | Blue | Intermediário | Design systems, contexto, iteração, tweaks |
| T3 | Avançado & Expert | Purple | Avançado | System prompt, hacks, starter components, verificador |
| T4 | Galeria claude.ai/design | Amber | Todos | Telas reais, cases Canva / Brilliant / Datadog, showcases |
| T5 | Prompts & Automações | Teal | Avançado | Prompts-campeões, handoff Claude Code, workflows design→dev |

**Total:** 5 trilhas × ~7 módulos = **~35 módulos** de conteúdo, com ilustrações e exemplos reais.

## Como usar

Abra `index.html` no navegador, ou acesse via GitHub Pages (em breve).

Cada trilha tem:
- Um `index.html` com cards expansíveis dos módulos
- Uma página `modulo-X-Y.html` por módulo com conteúdo completo (6–8 tópicos, dicas, grids fazer/não-fazer, timelines, resumo)

## Padrão visual

Todas as páginas seguem o template **INEMA.CLUB**: navegação global com logo amarelo + link sky-400, cards de módulos com tópicos expansíveis de 3 seções (o-que-é / por-quê / conceitos-chave), light/dark mode, sem gradientes no light mode, botões sempre `justify-start`.

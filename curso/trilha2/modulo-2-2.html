📚 Anatomia de um context stack rico

1

🧱 O que é context stacking

A técnica mais importante da trilha 2: você empilha múltiplas fontes ao invés de escolher uma. Claude cruza todas e extrai padrão — resultado mais rico que qualquer fonte isolada.

🔬 Por que funciona

Cada fonte cobre um ângulo diferente: DESIGN.md diz como deve parecer; PDF diz o que precisa ser dito; URL diz quem é a referência; screenshot diz o vibe. Junto: contexto 360°.

2

📄 PDF longo como input

Relatórios, papers, e-books, transcrições. Claude invoca nativamente a skill read_pdf — sem pré-processamento.

Prompt-receita: PDF → deck

"Leia [arquivo.pdf]. Extraia as 10 ideias mais importantes.
Transforme em deck de 10 slides (1 ideia por slide).
Use deck_stage.js. Cada slide com visual impactante —
evite slides com muito texto."
3

🌐 Site como referência: capture > screenshot

Web capture retorna texto extraído, não HTML nem layout. Para "design como este site", complemente com screenshot — Claude então vê o visual e lê a estrutura.

✗ Só URL

Claude extrai palavras, não layout. Copy fica similar, visual não.

✓ URL + screenshot

Palavras + visão. Claude copia estrutura visual E tom editorial.

4

📋 Texto bruto: raw > structured

Pare de pré-processar. Cole transcrição de entrevista, notas bagunçadas, brainstorm em lista. Claude estrutura melhor que você.

⚡ Hack

Peça para Claude estruturar ANTES de gerar: "Leia a transcrição. Me liste as 5 ideias principais em bullets curtos. Depois disso, eu escolho qual vira design."

5

🗜️ Snip automático: Claude compacta sozinho

Cada mensagem carrega [id:mNNNN]. Quando uma fase termina, Claude usa tool snip para marcá-la para remoção. Executa quando contexto aperta — silencioso.

Como é, na prática

6

🎯 Ordem importa: DESIGN.md primeiro

Sequência ideal: (1) DESIGN.md (ancora estética), (2) URL/Figma de referência (reforça visual), (3) conteúdo (PDF, texto), (4) prompt final. Inverter a ordem confunde Claude.

🧠 Por quê?

Claude constrói o "mental model" à medida que lê. Se você dá conteúdo bruto antes da estética, ele começa a imaginar "como esse texto ficaria bonito" — e pode divergir do seu DS.

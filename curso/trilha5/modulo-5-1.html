MÓDULO 5.1

🎯 Prompt-campeão: Pitch deck

Template pronto para seed round. Copie, personalize os [colchetes], cole no Claude Design.

📋 Template pronto para colar

Crie um deck de 10 slides para seed round de [produto/empresa],
vertical [SaaS B2B / fintech / healthtech / etc].

AUDIÊNCIA: [sindicato de anjos tech / VC seed fund / angel investor group]
TAKEAWAY CENTRAL: [a mudança que você causa — ex: "prep de auditoria
  de 6 semanas para 4 dias"]

ESTRUTURA (1 ideia por slide):
1. Title: nome + 1 linha do que faz
2. Problem: dor do mercado, quantificada
3. Solution: como você resolve, 1 frase
4. Market: tamanho, crescimento (TAM/SAM/SOM)
5. Product: 3-4 features principais com visual
6. Biz Model: como ganha dinheiro
7. Traction: números reais (MRR, users, logos)
8. Team: quem + por quê vocês
9. Ask: quanto, no que será usado (18 mo runway)
10. Vision: onde isso nos leva em 5 anos

VISUAL:
- Feeling: [Stripe × A24 / Linear / Warm editorial / Cinematic dark]
- Restrições: NO newspaper-minimalism, NO gradient blobs,
  NO floating orbs, NO Inter/Roboto fonts
- Tipografia: [Newsreader italic + Geist] ou [Söhne]
- Palette: [primary #hex + neutral #hex + accent #hex]

TÉCNICO:
- Use deck_stage.js
- 1920×1080 fixo
- Include speaker notes (full conversational script per slide)

EXPORT FINAL: PDF para investor share + PPTX para editar
1

📋 Como usar o template

Copie o bloco acima. Substitua os [colchetes] com seus dados. Cole no Claude Design. Ele vai pular questions_v2 porque tudo está front-loaded.

2

📎 Hack: anexar PDF bruto

Em vez de escrever conteúdo nos bullets, anexe PDF do one-pager, data room, memo. Claude extrai as ideias-chave e visualiza. Funciona EXTREMAMENTE bem.

// Adicione ao template acima:
"Read the attached [one-pager.pdf]. Extract
the key ideas and map to the 10-slide structure.
Simplify complex information. Each slide = 1 idea."
3

🎨 Remix de estilos visuais

O mesmo conteúdo pode ir para VCs diferentes com estéticas completamente diferentes. Só mude a linha "Feeling":

SAFE · CORPORATE

"Feeling: Linear professional, clean serif, accent azul escuro"

BOLD · STARTUP

"Feeling: Stripe × A24, oversized type, cinematic dark moments"

HUMAN · CONSUMER

"Feeling: Warm editorial like Anthropic, coral + cream, italic serif"

TECH · DEV

"Feeling: Terminal-core (Ollama/Warp), mono font, verde/amber em dark"

4

🔊 Speaker notes como roteiro

Adicione ao template: "Include speaker notes: full conversational script per slide, ~90-120 seconds each, first person." Resultado: roteiro completo para falar 15-20 minutos sem improvisar.

5

📊 Charts de dados via recharts

Em slides de Traction, peça charts interativos. Passe CSV inline:

"Slide 7 (Traction): use recharts via CDN for growth chart.
Data (MRR in $k):
Jan: 12, Feb: 18, Mar: 28, Apr: 45,
May: 62, Jun: 89, Jul: 128, Aug: 184.
Line chart, primary color accent."
6

📤 Exports combinados

Do mesmo projeto, você gera: PDF (para imprimir e entregar), PPTX (para editar no PowerPoint/Google Slides), Share URL interna (para sócios comentarem). Segundos cada.

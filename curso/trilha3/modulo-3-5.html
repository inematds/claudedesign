🎭 Anatomia de uma cena animada

1

🎭 Stage: o container

<Stage duration={ms}> envolve toda a animação. Fornece autoscale responsivo, scrubber timeline, play/pause nativos.

Props úteis

  • • duration: duração total em ms (obrigatório)
  • • autoPlay: default true
  • • loop: para loops infinitos
  • • scrubber: false oculta o timeline (para export vídeo)
2

✨ Sprite: o elemento animado

<Sprite start={N} end={M}>...</Sprite>. Existe só entre start e end (em ms). Antes de start: hidden. Depois de end: hidden.

Composição em cenas

Múltiplos Sprites dentro do Stage formam uma cena. Sobreposição de intervalos = elementos coexistem em tempo. Use isso pra criar ritmo.

3

⏱️ useTime + useSprite: hooks

useTime() retorna o tempo absoluto do Stage (em ms). useSprite() retorna o progresso 0-1 do Sprite atual (0 no start, 1 no end).

function Hero() {
  const progress = useSprite();  // 0 a 1
  return (
    <div style={{
      opacity: progress,
      transform: `translateY(${(1-progress) * 40}px)`
    }}>
      Seu produto em uma linha
    </div>
  );
}
4

📈 Easing + interpolate

Easing.easeOut(v) aplica curva ao valor linear 0-1. interpolate(v, [0,1], [0,100]) mapeia range.

// Easings disponíveis
Easing.easeIn, Easing.easeOut, Easing.easeInOut,
Easing.easeBack, Easing.easeBounce, Easing.linear

// Uso combinado
const p = useSprite();
const y = interpolate(Easing.easeOut(p), [0, 1], [40, 0]);
// y = 40 → 0, acelerando no começo e desacelerando no fim
5

🚪 Entry/exit primitives

Helpers prontos para padrões comuns: fadeIn, fadeOut, slideInUp, slideOutDown, scaleIn, scaleOut. Cobrem ~80% das animações de entrada/saída.

6

🎁 Popmotion só como último recurso

System prompt explícito: "Only fall back to Popmotion if the starter genuinely can't cover the use case." CDN específico: unpkg.com/popmotion@11.0.5/dist/popmotion.min.js.

⚠️ Consumo de quota

Geração de animações consome tokens rapidamente. No plano Pro, 2-3 sessões de animação intensa podem esgotar a cota semanal. Use Max se for rotina.

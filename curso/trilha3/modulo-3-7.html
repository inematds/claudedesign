🧙 Manipulação avançada do system prompt

⚡ Hierarquia: user > system

1

⚡ Princípio: user > system (exceto ética)

Anthropic garante: user prompt sempre tem prioridade sobre system prompt, desde que não cruze a linha ética (IP de outras empresas, conteúdo ilegal, etc). É o que permite TODOS os hacks a seguir.

2

❓ "Skip the 10 questions"

Use quando você sabe o que quer e não tem tempo. "Skip questions_v2. Use these defaults: [lista]. Generate now."

⚠️ Risco

Skipping sem contexto → newspaper default garantido. Só funciona bem SE você também passa DESIGN.md + referências.

3

🔢 "Me dê N variações"

Default é 3. Você pode pedir até 7 com bom resultado. Acima de 7, Claude começa a repetir conceitos.

"Give me 6 variations. Start safe (match my DS 100%)
 and progress to novel (V6 can experiment with layout
 and motion). Axis: 2 color, 2 type, 2 density."
4

🚫 "No newspaper-minimalism"

Uma frase mata o viés embutido. Combine com feeling alternativo para direcionar.

"NO newspaper-minimalism. Specifically:
 - NO italic serif headlines
 - NO off-white backgrounds
 - NO pull-quote sections labeled editorial
 - NO 'Our Portfolio' style section titles
 Instead, aim for: Cinematic Dark (A24-inspired),
 oversized type, generous spacing."
5

🛑 "No AI tropes": a anti-slop frase

Os 4 grandes que denunciam output AI-gerado. Sempre inclua no prompt se quer originalidade:

"NO AI design tropes:
 - NO gradient blobs or mesh gradients as background
 - NO floating orbs or glowing spheres
 - NO glassmorphism (frosted glass cards)
 - NO emojis outside of DS specification
 - NO Inter, Roboto, Arial, or Fraunces typography"
6

🎯 Força bruta: "Use exactly X, Y, Z"

Para marca muito definida — zero improvisação criativa. Liste cada token:

"Use EXACTLY these tokens (no improvisation):
 Primary: #E07A4F (coral). Secondary: #F4F0EA (cream).
 Neutral: #0a0a0f (near-black).
 Headline: Newsreader italic 500.
 Body: Geist Regular 400.
 Spacing scale: 4, 8, 16, 24, 32, 48, 64 (no others).
 Radii: card 12px, button 6px, chip 999px.
 Shadow: single-level (0 2px 8px rgba(0,0,0,0.08))."
